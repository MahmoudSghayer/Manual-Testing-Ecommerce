# E-Commerce Application – Functional Requirements

## R1 – User Registration
- The system shall allow users to register using a valid email and password
- The system shall display validation messages for missing or invalid registration fields
- The system shall prevent registration with invalid email format

## R2 – User Login
- The system shall allow registered users to log in with valid credentials
- The system shall block login attempts with invalid credentials
- The system shall validate empty email and password fields

## R5 – Shopping Cart
- The system shall allow users to add products to the shopping cart
- The system shall update cart quantity and total price correctly
- The system shall allow users to remove products from the cart
