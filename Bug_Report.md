# Bug Report

## Bug ID
BUG-001

## Title
Registration allows invalid email format in some cases

## Module
User Registration

## Environment
- Browser: Chrome
- OS: Windows 10
- Testing Type: Manual UI Testing

## Precondition
User is on the registration page.

## Steps to Reproduce
1. Open the registration page
2. Enter an invalid email format (e.g. user@com)
3. Fill other mandatory fields with valid data
4. Click Register

## Expected Result
System should display an error message indicating invalid email format and block registration.

## Actual Result
System allows registration attempt or displays inconsistent validation behavior.

## Severity
Medium

## Priority
Medium

## Status
Open

## Reported By
QA Tester

## Notes
This bug is reported for demonstration and portfolio purposes.
